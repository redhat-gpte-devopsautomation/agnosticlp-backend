== OAuth 2.0 Terminology

* Resource owner (user)
* Client (Yelp app)
* Authorization server (Google authorization server)
* Resource server (Google API server)
* Authorization grant (User says 'Yes')
* Redirect URL (callback)
* Access token (Yelp needs key to access data)
